## 📌 What is GlowCity?

GlowCity Mumbai is an AI-powered beauty salon marketplace
built for Mumbai's urban beauty seekers. Discover, compare,
and instantly book top-rated salons, spas, barbers, and
bridal studios across 30+ Mumbai areas — all in one place.

No app download. No phone calls. Just book and glow.

---

## ✨ Key Features

- 🔍 **Smart Search** — Filter by area, service, price & rating
- 🤖 **AI Beauty Concierge** — Describe what you need in plain
  language, get instant personalised salon recommendations
- 📅 **Instant Booking** — Pick service, date, time slot & confirm
  in under 30 seconds
- 🏪 **Partner Dashboard** — Shopkeepers can list, manage and
  track their salon listings
- 🌙 **Dark / Light Mode** — Persistent preference saved via
  localStorage, system preference detected on first visit
- 📱 **Mobile First** — Fully responsive across all screen sizes
- ⭐ **Verified Reviews** — Real ratings from real customers
- ⚡ **Zero Dependencies** — No npm, no frameworks, no build step

## 🤖 AI Feature — Beauty Concierge

The standout feature of GlowCity is the **AI Beauty Concierge**
— a sticky, collapsible chat widget available on every page.

Users can type something like:

> *"Bridal makeup under ₹5000 near Bandra"*
> *"Cheap men's haircut in Dadar"*
> *"Relaxing couple's spa day in Juhu"*

The concierge uses natural language understanding to return
personalised salon recommendations tailored to the user's
need, budget, and Mumbai area.

**UI/UX highlights of the AI widget:**
- Distinct message states (`.ai-msg` vs `.user-msg`)
- "AI thinking" animation with CSS keyframes
- Radar pulse (`magicPulse`), comet tail (`sparkSpin`),
  and dancing icon (`sparkleDance`) micro-interactions
- Fully collapsible and mobile friendly

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Flexbox, Grid, Keyframes) |
| Logic | Vanilla JavaScript ES6+ |
| Icons | FontAwesome v6.0.0 (CDN) |
| Typography | Google Fonts — Playfair Display + Plus Jakarta Sans |
| Browser APIs | IntersectionObserver, localStorage, DOM Manipulation |
| Hosting | GitHub Pages |

> 💡 **Zero npm dependencies.** No React, no Tailwind, no
> build tools. Pure vanilla web tech — resulting in
> near-instant load times and maximum performance.
